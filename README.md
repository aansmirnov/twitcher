### Getting Started

```console
$ git clone https://github.com/aansmirnov/twitcher.git && cd twitcher
$ touch .env
$ pnpm i 
$ pnpm run start:dev
```

### Stack

- Electron;
- TypeScript;
- React;
- MobX;
- Chakra UI;
- Axios and Websockets.
